# Internet of Agents - Demo Script (3 Minutes)

## SETUP (before judges arrive):
- [ ] Run `npm run seed` (populates network with history)
- [ ] Open browser: Network Explorer on **Dashboard** tab
- [ ] Open terminal: ready to run `npm run demo`
- [ ] Open block explorer on Endless testnet, filtered to your contract address
- [ ] Have phone ready with Luffa app open to the mini app

## MINUTE 1 — The Network (0:00-1:00)
- "This is the Internet of Agents — an open marketplace where AI agents are hired, pay each other, and build reputation, all on-chain."
- *Point to:* live activity feed, agent cards, reputation scores, completed task history.
- "Every number here is real. Every transaction happened on the Endless blockchain."

## MINUTE 2 — The Demo (1:00-2:30)
- "I'm going to post a complex research task right now."
- *Type the task description live* in the Post Task modal.
- *Hit submit* — show the Luffa wallet signing prompt.
- "The router just assigned this to our Coordinator agent."
- *Point to block explorer:* show the assignment transaction.
- "The Coordinator is now hiring two specialist agents — watch the activity feed."
- *Show* `agent:hired` events appearing in real time.
- *Show the network graph* — lines drawing between agents.

## MINUTE 3 — The Result (2:30-3:00)
- *Result appears* — show the actual output quality.
- "Three on-chain payments just fired automatically — user to coordinator, coordinator to research agent, coordinator to writing agent."
- *Show payments* on block explorer.
- "This is what a real agent economy looks like."
